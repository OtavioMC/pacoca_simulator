# 🥜 Vendedor de Paçoca Simulator

> *“Do fundo do poço ao primeiro milhão… talvez.”* 🚀

Um simulador tragicômico de sobrevivência urbana onde você tenta escapar da pobreza extrema vendendo paçoca na rua.

Gerencie fome, banho, cansaço, dívidas, financiamentos absurdos, assaltos, abordagens policiais e discursos motivacionais completamente duvidosos.

Seu objetivo principal é simples:

# 💰 VIRAR MILIONÁRIO

---

## 🎮 Sobre o Projeto

Este jogo é uma demonstração desenvolvida para a disciplina de Jogos Digitais do CEFET-RJ.

O projeto mistura:
- gerenciamento de recursos;
- narrativa satírica;
- humor absurdo;
- crítica social;
- mecânicas de progressão econômica extremamente brasileiras.

---

## 🥜 Gameplay

Você começa:
- sem dinheiro;
- morando em uma barraca furada;
- sem comida;
- sem banho;
- e com um coach dizendo que “o sucesso depende apenas de você”.

A partir daí, você deve:
- pedir dinheiro;
- vender paçoca;
- sobreviver ao dia;
- comprar upgrades;
- evitar ser preso;
- financiar imóveis impossíveis;
- e tentar enriquecer antes que sua vida desmorone.
